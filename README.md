# Performance-Analyser-Feedback-System
Performance Analyser &amp; Feedback System – A Streamlit-based application that analyzes user performance data using Pandas and Python, generates visual insights, and provides AI-powered feedback using LLMs.
Detailed Project Description
Performance Analyser & Feedback System

Developed an intelligent performance analysis application using Python, Pandas, Streamlit, and Large Language Models (LLMs). The system processes user performance datasets, performs data cleaning and statistical analysis, visualizes key metrics through interactive dashboards, and generates personalized feedback using AI.

Features
📊 Interactive performance dashboard with Streamlit
🧹 Data preprocessing and analysis using Pandas
📈 Visualizations for trends, KPIs, and performance metrics
🤖 AI-generated performance feedback using LLMs
📂 CSV/Excel file upload support
💡 Actionable recommendations based on analyzed data
⚡ Real-time insights and user-friendly interface
Tech Stack
Python
Streamlit
Pandas
Matplotlib / Plotly
OpenAI / LLM APIs
